# 🛒 FreshCart – Online Grocery Shopping Website

FreshCart is a modern and user-friendly online grocery shopping platform inspired by Blinkit. It is designed to provide a seamless and interactive shopping experience, allowing users to browse products, add items to the cart, and manage their purchases efficiently.

## 🚀 Features

- 🛍️ Product listing with clean card layout  
- 🔍 Live search and filtering functionality  
- 🛒 Add to cart / remove from cart  
- 💰 Real-time price calculation  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ⚡ Dynamic UI updates without page reload  
- ✅ Form validation for better user input handling  

## 🧰 Tech Stack

- HTML5 – Structure and layout  
- CSS3 – Styling and responsiveness  
- JavaScript – Interactivity and dynamic features  

## 🎯 Project Objective

The goal of FreshCart is to replicate the experience of modern grocery delivery platforms like Blinkit by focusing on:
- Simple navigation  
- Fast interactions  
- Clean and attractive UI  

## 🧠 Key Learnings

- DOM manipulation using JavaScript  
- Handling user interactions and events  
- Building responsive layouts using CSS  
- Implementing cart logic and dynamic updates  

## 📌 Future Improvements

- User authentication (login/signup)  
- Backend integration (Node.js / Firebase)  
- Payment gateway integration  
- Order history and tracking
